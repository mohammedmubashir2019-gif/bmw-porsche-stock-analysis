# BMW vs Porsche Stock Market Analysis

## Overview
This project analyzes and compares BMW and Porsche stock market data using Python, Pandas, Matplotlib, and SQL.

The project includes:
- Data cleaning
- Exploratory data analysis
- Correlation analysis
- Data visualization
- SQL-based stock queries

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- SQL
- VS Code

---

## Dataset Features

The datasets contain:
- Open price
- High price
- Low price
- Closing price
- Trading volume

---

## Analysis Performed

### Data Cleaning
- Removed missing values
- Standardized column names
- Cleaned stock datasets

### Statistical Analysis
- Average closing prices
- Highest and lowest prices
- Correlation between stock prices

### Visualizations
- BMW vs Porsche closing price comparison
- Daily returns graph
- Porsche closing price histogram
- Correlation heatmap

---

## SQL Queries Included

- Highest closing prices
- Lowest closing prices
- Average trading volume
- Price filtering
- Top 5 highest stock prices
