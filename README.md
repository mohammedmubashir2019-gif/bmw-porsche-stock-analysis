# BMW vs Porsche Stock Market Analysis

## Author
Mohammed Mubashir Ahmed

---

## Overview

This project analyzes and compares BMW and Porsche stock market data using Python, Pandas, Matplotlib, and SQL.

The project includes:

- Data cleaning
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Data visualization
- SQL-based stock queries

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- SQL
- VS Code
- Git & GitHub

---

## Project Features

### Data Cleaning
- Removed missing values
- Organized stock datasets
- Created cleaned CSV files

### Exploratory Data Analysis
- Stock price trends
- Average stock prices
- Trading volume analysis
- Daily returns calculation

### Data Visualization
The project generates:

- BMW closing price histogram
- Porsche closing price histogram
- BMW vs Porsche comparison chart
- Correlation heatmap
- Daily returns graph

### SQL Analysis
SQL queries included for:

- Highest closing prices
- Lowest closing prices
- Average trading volume
- Price filtering
- Top 5 highest stock prices
